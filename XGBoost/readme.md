# XGBoost Binary Classifier Example

A demonstration of XGBoost implementation for binary classification using synthetic data, showcasing model training, evaluation, and performance metrics.

## 📋 Overview

This project provides a clean, executable example of using XGBoost (Extreme Gradient Boosting) for binary classification tasks. It generates synthetic data with configurable parameters, trains an XGBoost classifier, and evaluates its performance using standard metrics.

## 🚀 Features

- Synthetic dataset generation with customizable parameters
- Train-test split for proper model evaluation
- XGBoost classifier with optimized hyperparameters
- Performance evaluation using accuracy and classification reports
- Ready-to-run code with minimal dependencies

## 📊 Model Performance

The trained model achieves:
- **Accuracy**: 88.67%
- **Precision (Class 0/1)**: 0.91 / 0.87
- **Recall (Class 0/1)**: 0.87 / 0.90
- **F1-Score (Class 0/1)**: 0.89 / 0.89
